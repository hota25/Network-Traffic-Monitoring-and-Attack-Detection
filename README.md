 Designed and implemented a Network Traffic Monitoring and Attack Detection system in a virtualized cybersecurity lab environment. The project captures, analyzes, and monitors network traffic in real time to identify malicious activities and security threats. By leveraging packet inspection, traffic analysis, and machine learning techniques, the system detects attacks such as DDoS, port scanning, brute-force attempts, ARP spoofing, and other anomalous network behaviors. The solution provides actionable security alerts, traffic visualizations, and threat reports to enhance network visibility and incident response.
